# вҡҷпёҸ Installation Guide

## рҹ”№ Clone Repository

```bash
git clone https://github.com/Tanyav-rshney/Mood-based-music.git
cd Mood-based-music
```

---

## рҹ”№ Backend Setup

```bash
cd backend

npm run dev
```

---

## рҹ”№ Frontend Setup

```bash

```
