# ⚙️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/Tanyav-rshney/Mood-based-music.git
```

## Backend Setup

```bash
cd backend
npm install
npm start
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
